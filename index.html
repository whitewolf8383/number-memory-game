Number Memory Game

Rules: This memory game is a spinoff from the traditional Simon game. To start the game, simply push the start button. The game will start by showing you a number. Click on the number and hit the submit button. If you're correct, it'll play the next number. You will need to repeat all numbers from the start of the game to successfully pass the current stage. The game does not repeat the prevous numbers, so you'll need to remember them. Try and make it as far as possible. A counter will keep track of how fgar you make it.
